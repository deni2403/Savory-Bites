# Savory-Bites
Restaurant Catalog using Progressive Web App (PWA)

# Live Demo :
https://savory-bites-lite.netlify.app/

# This project was built using :
- HTML
- CSS
- Javascript
- Webpack

# API used :
https://www.themealdb.com/api.php

## Instalation :
```bash
$ git clone https://github.com/username/project-name.git
$ cd project-name
$ npm install
```

## Development :
```bash
$ npm run start-dev
```

## Build :
```bash
$ npm run build
```
